## 🚀 What I Do

- **DevOps & SRE**:
  - Design and implement robust **CI/CD** pipelines using tools like **Jenkins**, **GitLab CI**, and **CircleCI**.
  - Build and manage scalable **cloud infrastructures** on **AWS**, **Azure**, and **GCP**.
  - Develop Infrastructure as Code (**IaC**) solutions with **Terraform** and **Ansible**.
  - Manage and deploy containerized applications using **Docker** and **Kubernetes**.
  - Implement and fine-tune **monitoring and alerting systems** with **Prometheus**, **Grafana**, and the **ELK Stack**.

- **Automation Testing**:
  - Integrate automated test suites (**Playwright**, **k6**, **Selenium**) into CI/CD workflows to ensure high software quality.
  - Develop scripts and automation frameworks in **Python**, and **Shell** for streamlined testing and operations.
